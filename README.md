# Cisco Network Design and Implementation

A complete enterprise network implementation using Cisco Packet Tracer with subnetting, DHCP, DNS, HTTP services, static routing, and network communication testing.

## Project Overview
This project was designed and implemented using Cisco Packet Tracer as part of a Computer Networks assignment.

The network includes multiple routers, switches, servers, and end devices connected using static routing and subnetting techniques.

## Features
- Subnetting and IP address allocation
- Static routing configuration
- DHCP configuration
- DNS configuration
- HTTP services
- End-to-end network communication testing
- Multiple network segments
- Enterprise-style topology design

## Network Components
- Routers
- Switches
- DHCP Servers
- DNS Server
- HTTP Servers
- PCs and Laptops

## Services Implemented
- DHCP for automatic IP assignment
- DNS for domain name resolution
- HTTP web services
- Static and default routing

The network topology image is included in this repository.

- Cisco Packet Tracer
- Networking Fundamentals
- IPv4 Subnetting
- Static Routing
